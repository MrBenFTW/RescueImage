Installation via Linux (Rescue OS)

  1. Boot the Rescue OS - Each provider is different. For OVH it is NetBoot and for Hetzner it is Rescue.
  2. Login Via SSH - When you have the connection details, login to your server over ssh and continue to step 3.
  3. Install Windows - Once you have logged in via ssh, run the following command: wget -O- 'https://cdn.pyron.tech/Images/Windows/WindowsServer2019.img.gz' | gunzip -c | dd of=/dev/sda
  4. Finish - now that the OS has been installed, you can finish off by running the "reboot" command. Open your remot desktop client and conenct to your server. Your Username is: Administrator And your password is Password998
